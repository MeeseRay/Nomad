# NOMAD | DEPLOYMENT/IMPLEMENTATION

### Requirements |

## Software |
### Godot v4.1.2
https://godotengine.org/download/archive/
# Windows |
### Minimum |
OS | Windows 7
Processor | Support for OpenGL 2.1
Memory | 2 GB
### Recommended |
OS | Windows 10 (or newer)
Processor | Support for OpenGL 3.3
Memory | 4 GB

## MacOS |
### Minimum |
OS | MacOS X 10.7 - Lion
Processor | Support for OpenGL 2.1
Memory | 2 GB
### Recommended |
OS | MacOS 11 - Big Sur
Processor | Support for OpenGL 3.3
Memory | 4 GB

## Instruction |
1. Click the link above to access te Godot archives
2. Scroll down until you see Godot v4.1.2
***Be sure to install the "stable" version for best results***
3. Download the Github file for Nomad
4. Open Godot, Click Import -> Browse and search for the file path of Nomad.
5. Choose the .godot file and click Open
6. Return to the project manager and click the file to run project.
